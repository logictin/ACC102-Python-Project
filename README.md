# ACC102 Python Data Analysis Project
## Project Topic
Analysis of China's total retail sales of consumer goods from 2021 to 2025

## Data Source
National Bureau of Statistics of China

## Files Included
- analysis.py: Python code for data analysis and visualization
- 年度数据.xlsx: Raw dataset
- 零售总额分析图.png: Generated chart
- reflection.md: Reflection report
- demo_video.mp4: Demonstration video for Track 2

## How to Run
1. Install required libraries:
pip3 install pandas matplotlib openpyxl

2. Run the code:
python3 analysis.py
